## 🛠 Installation

Clone the repository:

```bash
git clone https://github.com/dhruvmahajan001/CodeAlpha.git
```

Navigate to a project:

```bash
cd CodeAlpha/task1
```

or

```bash
cd CodeAlpha/task2
```

Install backend dependencies:

```bash
cd backend
npm install
```

Create a `.env` file and configure:

```env
PORT=5000
MONGODB_URI=your_mongodb_uri
JWT_SECRET=your_jwt_secret
CLIENT_URL=http://localhost:5500
NODE_ENV=development
```

Run the backend:

```bash
npm run dev
```

Serve the frontend using Live Server or:

```bash
serve frontend -p 5500
```
